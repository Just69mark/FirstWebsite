Forms and Input

Introduction to HTML Forms

HTML forms are used to collect user input. They are an essential part of many websites, allowing users to submit data, log in, make purchases, and more.

Basic Form Structure


<form action="/submit-form" method="post">
    <!-- Form elements go here -->
</form>

Common Form Elements

  • <input>: Used for various input types (text, password, checkbox, radio, etc.)
  • <textarea>: For multi-line text input
  • <select> and <option>: For dropdown menus
  • <button>: For submit buttons and other clickable elements
  • <label>: To associate a label with an input element

Example: Creating a Simple Form


<form action="/submit-form" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>

    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>

    <label for="message">Message:</label>
    <textarea id="message" name="message" rows="4" required></textarea>

    <label for="country">Country:</label>
    <select id="country" name="country">
        <option value="usa">United States</option>
        <option value="canada">Canada</option>
        <option value="uk">United Kingdom</option>
    </select>

    <label>
        <input type="checkbox" name="subscribe" value="yes">
        Subscribe to newsletter
    </label>

    <button type="submit">Submit</button>
</form>

HTML5 Input Types

HTML5 introduced several new input types to make form handling easier:

  • email: For email addresses
  • url: For web addresses
  • number: For numeric input
  • date: For date pickers
  • color: For color selection
  • range: For slider controls

Form Validation

HTML5 also provides built-in form validation attributes:

  • required: Specifies that an input field must be filled out
  • min and max: Specifies the minimum and maximum values for an input field
  • pattern: Specifies a regular expression that the input field's value is checked against
