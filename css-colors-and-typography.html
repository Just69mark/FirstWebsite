Colors and Typography in CSS

Colors in CSS

Colors play a crucial role in web design. CSS provides several ways to specify colors:

1. Color Keywords

CSS provides a set of predefined color keywords.


color: red;
background-color: blue;

2. Hexadecimal Colors

A six-digit code representing RGB values.


color: #FF0000; /* Red */
background-color: #0000FF; /* Blue */

3. RGB and RGBA

RGB specifies Red, Green, and Blue values. RGBA adds an Alpha channel for opacity.


color: rgb(255, 0, 0); /* Red */
background-color: rgba(0, 0, 255, 0.5); /* Semi-transparent Blue */

4. HSL and HSLA

HSL specifies Hue, Saturation, and Lightness. HSLA adds an Alpha channel.


color: hsl(0, 100%, 50%); /* Red */
background-color: hsla(240, 100%, 50%, 0.5); /* Semi-transparent Blue */

Typography in CSS

Typography is the art of arranging type to make written language legible, readable, and appealing. CSS provides various properties to control typography:

1. Font Family

Specifies the font for an element.


font-family: Arial, sans-serif;

2. Font Size

Sets the size of the font.


font-size: 16px;
font-size: 1.2em;
font-size: 120%;

3. Font Weight

Sets how thick or thin characters in text should be displayed.


font-weight: bold;
font-weight: 700;

4. Text Alignment

Sets the horizontal alignment of text.


text-align: center;
text-align: justify;

5. Line Height

Sets the height of a line of text.


line-height: 1.6;
