Tables and Lists

HTML Tables

HTML tables allow you to arrange data into rows and columns. They are useful for displaying tabular data such as product information, schedules, or comparison charts.

Basic Table Structure


<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Row 1, Cell 1</td>
            <td>Row 1, Cell 2</td>
        </tr>
        <tr>
            <td>Row 2, Cell 1</td>
            <td>Row 2, Cell 2</td>
        </tr>
    </tbody>
</table>

Example Table

HTML Lists

HTML provides three types of lists: unordered lists, ordered lists, and description lists.

Unordered List

An unordered list is a collection of related items that have no special order or sequence. List items are typically displayed with bullet points.


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered List

An ordered list is a collection of related items that follow a specific sequence. List items are typically displayed with numbers.


<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Description List

A description list is used to display name-value pairs such as terms and definitions.


<dl>
    <dt>Term 1</dt>
    <dd>Definition 1</dd>
    <dt>Term 2</dt>
    <dd>Definition 2</dd>
</dl>
