CSS Layout

Introduction to CSS Layout

CSS layout is about the arrangement of elements on a web page. It involves controlling how elements are positioned relative to the viewport and to each other.

Key Concepts in CSS Layout

1. Display Property

The display property specifies how an element should be displayed.


display: block;
display: inline;
display: inline-block;
display: flex;
display: grid;

2. Position Property

The position property specifies the type of positioning method used for an element.


position: static;
position: relative;
position: absolute;
position: fixed;
position: sticky;

3. Float Property

The float property is used for positioning and formatting content.


float: left;
float: right;
float: none;

Layout Techniques

1. Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns.


.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

2. CSS Grid

CSS Grid is a two-dimensional layout system for the web.


.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

3. Float Layout

While less common in modern web design, float can still be useful for certain layouts.

    float: left;
    width: 50%;
}
.right {
    float: right;

Responsive Design

Responsive design ensures that web pages render well on a variety of devices and window or screen sizes.

Media Queries


@media screen and (max-width: 600px) {
    .column {
        width: 100%;
    }
}
