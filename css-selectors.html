CSS Selectors

Introduction to CSS Selectors

CSS selectors are patterns used to select and style HTML elements. They are a fundamental part of CSS and allow you to apply styles to specific elements or groups of elements on your web page.

Basic Selectors

1. Element Selector

Selects all elements with the specified tag name.


p {
    color: blue;
}

2. Class Selector

Selects all elements with the specified class attribute.


.highlight {
    background-color: yellow;
}

3. ID Selector

Selects the element with the specified id attribute.


#header {
    font-size: 24px;
}

Combinators

1. Descendant Selector

Selects all elements that are descendants of a specified element.


div p {
    margin-left: 20px;
}

2. Child Selector

Selects all elements that are the immediate children of a specified element.


ul > li {
    list-style-type: square;
}

Pseudo-classes and Pseudo-elements

1. Pseudo-class

Selects elements based on a certain state.


a:hover {
    color: red;
}

2. Pseudo-element

Selects and styles a part of an element.


p::first-line {
    font-weight: bold;
}

Attribute Selectors

Selects elements based on their attributes or attribute values.


input[type="text"] {
    border: 1px solid #ccc;
}
